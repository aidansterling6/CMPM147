Experiment 04: Infinite Worlds

Description

In this experiment, I was trying to make infinite, generative worlds.

Technical

All three sketches use the same global seed. The location of the camera is also global, so the three sketches show the exact same scene but at different times of the year. Each tree rotates around its base using summed noise functions based on millis(). I give them a slow, wide sway; then I add a smaller, faster sway on top of that. The positioning of the tiles also uses noise. I use linear noise functions to get a value for each tile, then I choose what to draw on that tile based on what the value is. For example, if the value is low enough, I draw water. Each grass tile has a chance to spawn a tree, I use the hashing function given to use to do that. The textures of each tile are stored in images using createGraphics, just like the trees. I create multiple variations to break up the texture. For all of the tile textures, I draw an array of points on them, each with a different color. For grass, I extend the points upward a random amount. I also separated the top and bottom to make the trees fit in properly. For Dirt, I make each point a randomly shaped ellipse to make it look lumpy. For water, I use horizontally stretched, layered noise for the base. I then add a random value on top of that to break it up and remove seams. I then just change the colors based on the season. Dirt has some snow added to it using noise for winter, and the darker water zone is removed for ice.
