Description

For this experiment, the goal was to create an evolutional system that converges on an image. I was trying to get a very detailed final output with color.

Technical

This one was pretty simple. I changed the p5 version that was running because the default was much slower than more recent versions. I copy pasted the code for the slider bars and made two new ones. I am just creating 1000 rectangles and mutating their position, color, and alpha. I am also using one of the sliders to determine how many mutations to make. I am using randomGaussian for mutations.
