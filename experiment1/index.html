Experiment 1 - Glitch Code Generator

Hello!

Description

Create an interesting text generator on glitch.com and move it to my portfolio website

Technical

The given code replaces keywords with a random replacement and repeats this until no keywords exist. I wrote code and mixed and matched different lines to allow for randomness wile still maintaining proper syntax and logic.

Reflection

I thought that this was interesting, and I may want to look into it further in the future. I could make it so that objects don't have to just follow physics and could instead have other behaviors. I could also lean into nesting different behaviors. For example, creating a for loop and generating other code inside it. I had some bugs, but it was relatively straightforward to implement.
